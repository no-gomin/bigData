# 변수 조회
ls() 히든변수를 제외한 변수목록
ls(all.names = TRUE) 히든변수 포함한 변수목록
ls.str() 변수이름과 구조를 함께






## 출력
print() : 하나의 객체만 출력
cat() : 여러 객체를 출력 (행렬이나 리스트 등 복합 데이터 구조는 출력 불가)
	cat("result = ", result)
paste() : 출력문은 아님. 하나의 객체로 만들어 주는 역할
	paste(c(1,2,3), c("하나","둘","셋"), sep="은 ", collapse=";")




# 변수 삭제 
rm(a) : 특정 변수 삭제
rm(list=ls()) : 일반 변수 목록 일괄 삭제
rm(list=ls(all.names = TRUE)) : 히든 변수까지 모두 일괄 삭제
