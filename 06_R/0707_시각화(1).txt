## 그래프 함수

고수준 : 함수 호출할 때마다 새로운 그래프를 시작

저수준 : 새로운 그래프를 생성하지 않고, 이미 그려진 그래프 위에 점, 선, 글씨 등을 표시





## 그래프 파라미터 : 그래프를 그리는 함수들은 다양한 파라미터가 존재. par()함수를 통해 변경

par() : 그래프를 조정하거나 특성을 지정  ex. bty(박스 유형), 선 굵기, 색상...
-> park()함수의 리턴값은 실행전 특성 리턴

oldPar <- par(bty='L') # 파라미터 지정전 객체를 저장 
par(oldPar) # oldPar 설정으로 복귀

xlim, ylim : x, y축 눈금 조정
yaxt : y축 눈금 없애기
bty : 그래프 박스 type : o, ], L, 7, C

mtext() : 테두리 밖에 문자 추가

axis() : 축 정보를 추가 (1 하 / 2 좌 / 3 상 / 4 우)






## plot : 산점도 함수

oldPar <- par(mfrow=c(2,3)) # 그래프를 2행3열 (총6개) 생성하도록...

plot(cars, type="p", las=1)
# type : p 일반적 산점도 / l 추세선 / b 선과 점 겹치지 않게 / s 계단 형태 / n 안보이게 / o 선과 점 겹치게
# las : 축눈금라벨방향 -> 0 평행 / 1 가로 / 2 수직 / 3 세로
