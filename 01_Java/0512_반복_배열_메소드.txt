## while문, do-while문

while : 조건이 참일 경우 반복하여 수행 (*거짓인 경우 while문 탈출)

do - while : 최초 한번 무조건 실행 + 두번째부터는 while과 동일하게 수행

break; : 특정 조건을 만난 경우 반복문을 탈출

continue; : 특정 조건을 만난 경우 남은 명령문을 수행하지 않고, 반복문의 위로 올라가 수행





## 배열 (array) : 동일 자료형의 집합 (하나의 이름으로 다수의 데이터를 사용)

배열의 크기는 최초에 한번 설정되면 변경이 불가

int[] iArr = {10, 20, 30, 40, 50};        // 1.배열 변수 선언과 초기화
int[] iArr2 = new int[5];           // 2.배열변수선언과 배열메모리 공간확보(배열 객체 생성)
int[] iArr3;                             // 3. 배열 변수만 선언

for(int temp : iArr3) {
	System.out.println(temp);   }      // 배열의 경우는 확장for문으로 출력 가능

## 메소드(method) : 작업을 수행하기 위한 명령문의 집합 (어떤 값을 입력받아 처리후 그결과를 돌려줌)

public : 모두 접근 가능
protected : 동일한 패키지내에 존재하거나 파생클래스에서만 접근 가능
default : 아무런 접근 제한자를 명시하지 않은 경우, 동일한 패키지 내에서만 접근 가능
private : 자기 자신 클래스 내에서만 접근 가능
