## 접근지정자

public : 다른 "모든" 클래스에서 사용 가능

protected : "동일 패키지"의 클래스와 "상속받은" 클래스에서 사용 가능	

default : "동일 패키지"의 클래스에서 사용 가능

private : "해당 클래스"에서만 사용 가능





## static(정적변수, 공용변수) & final(종단변수)

static : 클래스 멤버라고 불림 ____&____ 클래스당 하나 생성 ____&____ 동일한 클래스의 모든 객체들에 의해 공유되어 사용
public static int count = 100;      // 생성된 인스턴세의 수를 저장하기 위한 static 변수

final : 종단변수로 상수값을 변경하지 못하게 한다.
final double PI = 3.141592;     // final변수인 PI 상수. 값을 변경하지 못한다. 전부 대문자로 만든다.
