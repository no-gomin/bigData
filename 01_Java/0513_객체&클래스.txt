## 객체지향 프로그램의 이해

객체 : 같은 종류의 데이터와 메소드가 함께 있는 구성체 (EX. 자동차)

클래스 : 객체를 만들기 위한 일종의 설계도

하나의 클래스에스 여러개의 객체를 생성할 수 있으며, 그객체들은 모두 독립적 존재




## 클래스의 기본적인 코딩방법

①패키지명   ②클래스명   ③데이터   ④생성자함수   ⑤메소드   ⑥Getter & Setter

단 하나의 생성자도 생성하지 않을시, default 생성자가 컴파일러 단계에서 자동으로 생성

매개변수가 있는 생성자를 생성시, 매개변수 없는 생성자를 생성하여 주는 것이 좋음

this.age = age;      // this : 객체자신의 (객체내 age변수에 age를 입력)
