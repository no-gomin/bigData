## 선택자 : HTML 문서 중 특정 부분을 선택하여, 해당 부분의 속성을 변경할 목적으로 사용

태그선택자 : 특정 태그를 선택

전체(*) 선택자 : * 페이지 초기화 용도로 사용

id(#)선택자 class(.)선택자 : id는 하나만, class는 중복선언 가능

속성 선택자 : ^='a' a로 시작 // $='a' a로 끝

후손선택자, 자손선택자 : >를 넣으면 자손(자식) // 후손은 자손을 포함

동위(형제)선택자 : +를 넣으면 태그 바로 뒤에 나오는 형제 하나 // ~는 태그 다음 나오는 모든 형제들

반응선택자 : :hover 마우스 hover 상태

상태선택자 : :disabled :enabled (ex.input텍스트 중 수정가능한? 불가능한?)

구조선택자 : :nth-child(2n+1) 몇(홀수)번째 // :first-child 첫번째 // :last-child 마지막
